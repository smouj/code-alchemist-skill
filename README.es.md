# Code Alchemist

[![ES](https://img.shields.io/badge/ES-Español-red)](README.es.md)

Habilidad Code Alchemist para OpenClaw.

## Qué soluciona
Transforma código desordenado en arquitecturas limpias y mantenibles mediante refactorización automatizada.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales y listos para producción
- Enfoque de seguridad primero
- Soporte para revertir cambios

## Uso
/code-alchemist

## Ejemplo
```
Analiza código para detectar problemas de codificación.
```

## Instalación
Esta habilidad está disponible automáticamente en OpenClaw.
